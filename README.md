# SHOOKACH

Simple gem for basic full-text search 

### Installation

```sh
$ gem install shookach
```

Or with Bundler in your Gemfile.

```ruby
gem 'shookach'
```

### Getting started

Indexing files in default directory

```ruby
Shookach.index
```

Indexing files in specific directory

```ruby
Shookach.index('path/to/dir/')
```
